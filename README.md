# QR Check-in Staff PWA

這是一個專為員工掃描 QR Code 簽到的 Progressive Web App (PWA)。

## 功能特色

- 📷 相機 QR Code 掃描
- 🔐 員工身份驗證
- 📊 簽到統計管理
- 🔄 離線支援與自動同步
- 📱 可安裝的 PWA 應用

## 使用方式

1. 點擊應用連結開啟
2. 使用員工帳號登入
3. 選擇要管理的活動
4. 開始掃描 QR Code 進行簽到

## 技術架構

- 純前端 PWA 應用
- 支援離線運作
- 連接到遠端 API: office.fanpokka.ai:8000
- 使用現代瀏覽器 Camera API

## 注意事項

- 需要允許相機權限才能掃描 QR Code
- 建議使用 HTTPS 環境以確保所有功能正常運作
- 支援安裝到手機主畫面作為原生應用使用
